# Calculator App

## Description
A desktop Calculator application developed using Python that performs basic arithmetic operations through a simple and user-friendly graphical interface.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear Screen
- Error Handling
- Responsive GUI

## Technologies Used
- Python 3
- Tkinter

## How to Run

1. Clone the repository
2. Install Python 3
3. Run

python app.py

## Future Improvements
- Scientific Calculator
- Dark Mode
- Keyboard Shortcuts
